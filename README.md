<h1 align="center" style="font-size: 50px;">osTicket - Prerequisites and Installation</h1>
<p style="font-size: 30px;"> This tutorial outlines the prerequisites and installation steps for setting up the open-source help desk ticketing system, osTicket. Follow this guide to install osTicket on a Windows 10 machine using Microsoft Azure, IIS, and MySQL. </p>

<h2 style="font-size: 40px;">Video Demonstration</h2>
<p style="font-size: 30px;"> - <strong><a href="https://youtu.be/9-Lji9myM-o">YouTube: How To Install osTicket with Prerequisites</a></strong> </p>

<h2 style="font-size: 40px;">Environments and Technologies Used</h2>
<ul style="font-size: 30px;"> 
<li><strong>Microsoft Azure</strong> (Virtual Machines/Compute)</li> 
<li><strong>Remote Desktop</strong></li> 
<li><strong>Internet Information Services (IIS)</strong></li> 
</ul>

<h2 style="font-size: 40px;">Operating Systems Used</h2>
<p style="font-size: 30px;"> - <strong>Windows 10</strong> (21H2) </p>

<h2 style="font-size: 40px;">List of Prerequisites</h2>
<ul style="font-size: 30px;"> 
<li>Install IIS (Internet Information Services)</li> 
<li>Install PHP Manager for IIS</li> 
<li>Install Rewrite Module</li> 
<li>Create PHP Directory</li> 
<li>Install Visual C++ Redistributable</li> 
<li>Install MySQL</li> 
<li>Enable Required PHP Extensions</li> 
</ul>
